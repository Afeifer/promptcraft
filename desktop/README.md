# PromptCraft Desktop

Portable Windows desktop app for creating perfect AI prompts.

## Development

```bash
npm install
npm start
```

## Build (portable .exe)

```bash
npm run build
```

The portable `.exe` will be in the `dist/` folder.

## Features

- Step-by-step prompt builder for Code, Text, Image, Video, Analysis, Translation, Learning, Chat, Pinterest
- AI-powered prompt enhancement via Gemini API (free)
- History & Favorites
- Multi-language UI (English, Russian, German)
- No installation required (portable)
