Privacy Policy

Last updated: May 2, 2025

PromptCraft ("we", "our", "the extension") is a Chrome browser extension that helps users create optimized prompts for AI models. This Privacy Policy describes how we collect, use, and protect your information.

1. Information We Collect

When you use PromptCraft, we may collect the following information:

  • Google Account Information (optional): If you choose to sign in with Google, we collect your name, email address, and profile photo. This is used solely for account identification and to enable features like data sync and update notifications.
  • Prompt History: Your generated prompts are stored locally on your device using Chrome's built-in storage API. This data never leaves your device unless you explicitly choose to share it.
  • Settings & Preferences: Your language preferences, API keys, and other settings are stored locally on your device.
  • Gemini API Key (optional): If you provide a Gemini API key for AI enhancement, it is stored locally on your device and sent only to Google's Gemini API for prompt enhancement requests.

2. How We Use Your Information

  • To provide and improve the extension's functionality
  • To authenticate your identity (if you choose to sign in)
  • To store your preferences and prompt history locally
  • To send you updates and news about new features (only if you opt in via newsletter consent)
  • To enhance prompts using the Gemini API (only when you explicitly request it)

3. Data Storage

  • Local Storage: Prompt history, favorites, settings, and API keys are stored locally on your device using Chrome's chrome.storage.local API.
  • Firebase/Firestore: If you sign in with Google, your basic profile information (name, email) and newsletter preference are stored in Google Firebase Firestore. This data is protected by Firebase Security Rules — each user can only access their own data.

4. Third-Party Services

PromptCraft uses the following third-party services:

  • Google Firebase Authentication — for Google Sign-In functionality
  • Google Cloud Firestore — for storing user profile data
  • Google Gemini API — for AI-powered prompt enhancement (only when you provide an API key and explicitly request enhancement)
  • GitHub API — for checking extension version updates

We do not sell, share, or transfer your personal data to any other third parties.

5. Data Security

We take reasonable measures to protect your information:

  • All communication with Firebase and Gemini API uses HTTPS encryption
  • Firestore Security Rules ensure users can only read and write their own data
  • API keys are stored locally and never transmitted to our servers
  • We do not have access to your prompt history or favorites — they are stored only on your device

6. Your Rights

You have the right to:

  • Access your data: View your stored data in the extension's History and Settings
  • Delete your data: Clear your prompt history, favorites, and settings at any time through the extension. To delete your Firebase account data, sign out and contact us
  • Opt out: You can use the extension without signing in. Sign-in is optional
  • Unsubscribe: You can opt out of newsletter communications at any time

7. Children's Privacy

PromptCraft is not directed at children under 13. We do not knowingly collect personal information from children under 13.

8. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of any significant changes through the extension or via email (if subscribed to updates).
